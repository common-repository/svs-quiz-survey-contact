== Description ==
SVS Quiz & Survey & Contact  - Visual Builder is a WordPress plugin that makes it easy to create and publish quizes, surveys and contact forms using a wizard and a visual builder. SVS Quiz & Survey & Contact works with any WordPress theme and you can publish your quizes/surveys/forms anywhere on your site using a shortcode. 


<p><strong>Features:</strong></p>
<p>- Create a quiz/survey/form using drag & drop</p>
<p>- See submissions in the admin</p>
<p>- Drag and drop & re-ordering</p>
<p>- Shortcode support</p>
<p>- Intuitive User Interface (visual builder)</p>

<p><strong>Available fields:</strong></p>
<p>- Layout / static text / header</p>
<p>- Text input / name / email / phone</p>
<p>- DropDown / Single Select / Multi Select</p>



== Installation ==
1. Upload wp_svs_quiz_survey_contact directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Create Pricing Tables using \"SVS Quiz\" menu in Wordpress Admin
